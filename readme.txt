Easy insertion of Hawaiian Diacriticals in TinyMCE.

== Description ==

This plugin adds a Hawaiian diacritical special character button to the WYSIWYG.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin through 'Plugins > Add New > Upload' interface or upload hawaiian_diacritics folder to the '/wp-content/plugins/' directory.
2. Activate the Hawaiian Diacritics plugin through the 'Plugins' menu in WordPress.
