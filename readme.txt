Easily check images in the Media Library for missing alt text.

== Description ==

This plugin adds the ability to find missing alt text of images in the Media Library. Having alternative text for images is imperative to be ADA and WCAG 2.0 compliant. Screen readers use this alternative text to describe images to the end user.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload the plugin through 'Plugins > Add New > Upload' interface or upload check_image_alt_text folder to the '/wp-content/plugins/' directory.
2. Activate the Check Image Alt Text plugin through the 'Plugins' menu in WordPress.
3. Click the Image Alt Text menu under the 'Settings' menu in WordPress.
